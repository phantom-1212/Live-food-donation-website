The **core layout component** where all major sections of the application are assembled to form the complete user interface.

### `index.css`
Defines:
- Global styles

- Typography
- Layout consistency

Configuration file used to:
- Enable smooth deployment on **Vercel**
- Handle client-side routing correctly
- Optimize production behavior

Contains the **minified and optimized production build** generated by Vite.  
This is the version of the app that is actually served to users in production.
